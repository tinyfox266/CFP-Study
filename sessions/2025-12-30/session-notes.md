**Concepts Covered**:
- Difference between lump sum and regular periodic payments

- Future Value of Annuity formula: FV = PMT × [(1 + r)^n - 1] / r
- Present Value of Annuity formula: PV = PMT × [1 - (1 + r)^-n] / r

### 5. Solving for r (Interest Rate) and n (Time)

**Concept**: Sometimes need to solve for unknown interest rate or time period

**Formula for n**: n = ln(FV/PV) / ln(1 + r)

**Formula for r**: r = (FV/PV)^(1/n) - 1

**Rule of 72**: Quick approximation for doubling time = 72 ÷ interest rate

**Status**: COMPLETED - Full comprehension verified

**Concepts Introduced**:
- Nominal Return = stated return (what you see)
- Real Return = return after inflation (what matters for purchasing power)
- Fisher Equation (Approximate): Real Return ≈ Nominal Return - Inflation Rate
- Fisher Equation (Exact): Real Return = [(1 + Nominal) / (1 + Inflation)] - 1
